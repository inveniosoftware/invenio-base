=====================
 Invenio-Base v1.0.0
=====================

Invenio-Base v1.0.0 was released on November 23, 2015.

About
-----

Base package for building the Invenio application.

*This is an experimental developer preview release.*

Installation
------------

   $ pip install invenio-base==1.0.0

Documentation
